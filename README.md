# Dao Unknown
# 恒道不知

把我对道的认识的核心概念和盘托出。很简单。不神秘。

我认为我所认知的道和我们科技时代的人类认知契合度，我不知道算不算完美，但至少极高。

同时，道是中国地域的认识，我们华人有充分的能力和必要去认识、演化、应用、传递道的解释体系。不仅让华人在科技时代处于世界的尖端，更是为人类在科技时代的演化提供多一种可选项。

而我本人对经济学人很敬佩和尊敬、对经济学人的解释体系定位也有过一点点粗陋的分析。我觉得道也应该有类似经济学人这样层次和制作水准的刊物。现在没有，将来也应该要有。这可以看成是道这一解释体系的追寻者们构建能力和形成为人类演化可选项之一的能力的一个指标。

也出于我自己对道和科学的认识，在我看来，华人中的自恨群体、对未来迷失群体、自大群体、自卑群体、唯这是从唯那是从的群体，都不知晓我们源于中国地域的人从先秦智慧思想中可继承和拥有什么样的思想工具箱，更不用说想到工具箱的应用可能性了。这样的子群体们有能力思考现实，并为华人的未来、人类的未来提供方向的可能性吗？

对未来，中国地域明摆着有自己的原生思维。我也远不是唯一或第一个看到或表达这点的人。实际上，我也是受我一位朋友的启发才有此发现的。

我希望至少在华人里，有越来越多的有识之士能了解和运用我们自己原生的思维工具看世界、参与世界、立足世界、助力世界。
